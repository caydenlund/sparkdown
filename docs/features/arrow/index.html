Arrows

Arrows are very useful in note-taking. Sparkdown makes it easy to write them.

The syntax is easy and intuitive:

ArrowResult
->
=>
<--
<==>

Control the direction with < and > characters. Control the width with - and = characters. Control the length with the number of - and = characters.
